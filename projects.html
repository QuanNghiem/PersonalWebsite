VR Engineering Education

Unity Application

Description:

VR Engineering Education is an application that replaces the need for buying and storing physical models by replicating said models within the virtual space. The application is tailored for engineering students, with the hope that by interacting with physics concepts and simulations, there would be an improvement in students' problems solving skills and understanding of physics in 3D space.

Github Link

Technolgy Used:

Unity Engine, Oculus Platform, SteamVR, C#, Blender, Unity Cloud.

Shifter

Unity Game

Trailer:

Description:

Battling robots and asteroids in the orbit of a deserted planet. Fight through waves of enemies to get back to Earth. Shifter is a game with the typical plot you see in most space arcade game. However, the player can switch from 2D to 3D to overcome obstacles. What inspired us to create that mechanic you ask? Absolute sheer boredom ... and a desire to learn more about camera and perspective manipulation in Unity.

Collabware

Android Application

Description:

CollabWare is a mobile app that has tinder-like functionality; you can press the heart button to connect with them, or x if you want to move on to the next person. However, instead of dating, the intention of this app is for software project ideas. A developer can input their idea in the bottom, as well as the languages that they are proficient in. If you press the heart button, they will show up in the chat page, and you can contact them to discuss about collaboration. Otherwise, if you click x, the app will show the next developer and their idea.

Github Link

Technolgy Used:

Dart, Flutter Framework, Android Studio, Github.
